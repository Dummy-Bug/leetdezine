# CAP Theorem — Overview

> You can have consistency and availability — until the network fails. Then you must choose.

> [!abstract] CAP theorem states that during a network partition, a distributed system can only guarantee two of three properties — Consistency, Availability, and Partition Tolerance. Since partitions are inevitable, the real choice is always CP or AP. This folder covers what that means, when to choose each, and how to apply it in every interview.
