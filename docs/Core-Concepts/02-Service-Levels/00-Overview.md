# Service Levels — Overview

> Without this framework, system health is subjective. With it, every decision is backed by a number.

> [!abstract] Metrics alone don't tell you if your system is doing well — you need targets, agreements, and consequences around those metrics. This folder covers the framework that turns raw measurements into actionable standards: SLI (what you measure), SLO (what you target), SLA (what you promise), and Error Budget (how much failure you can afford).
